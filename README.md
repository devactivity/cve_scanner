# cve_scanner

Belajar dasar pemrograman C, Haskell, Erlang, Rust

## Usage 

```bash
cd c_packet/
make

cd .. && cd erlang_manager
make # or
erlc -o ebin/ apps/erlang_manager/src/scanner.erl

cd .. && cd haskell_analyzer
cabal build

cd .. && cd rust_core
cargo build -r

cd .. && cd tui
cargo build -r
```
